## Usage Guidelines

### Figma Node IDs
Each component has a unique node ID. To get implementation details for any component:

```
Use MCP tool: get_design_context
Parameters:
  - nodeId: [component node ID from this index]
  - fileKey: 4Qnz6LBgqUdhDOE6SNrONX
```

### Example Usage
To implement a primary button:
1. Reference node ID: 57:1276 (State=Default, Type=Primary, Content=Text, Size=Regular)
2. Use get_design_context to fetch exact styling
3. Implement all required states (Default, Hover, Clicked, Disabled)
